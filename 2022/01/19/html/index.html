HTML标签

1.HTML：HyperText Markup Language 超文本标记语言

2.基本格式

<html>    //开始标签
    <head>
        <title>浏览器工具栏的标题</title>
    </head>

    <body>
    </body>
</html>   //结束标签
<!-- 注释 -->
<br/>   //换行标签
<p></p>   //段落标签 标签的开始和结尾默认产生一行空行。
<pre></pre>   //预设格式标记 按照原格式排版显示
<hr color="" width="" />   //水平线标签 颜色 宽度

字体标签

<font size="" color="">字体标签</font>
size 字号大小 最小值：1
              默认值：3
              最大值：7
<b>粗体</b>
<i>斜体</i>
<marquee>滚动字幕</marquee>

标题标签

图片标签

<img alt="" src="" />
alt:图片的文字说明
src:图片地址
width:图片显示的宽度
height：图片显示的高度

CSS 与 HTML 的结合方式

1、行内样式表

例: style="CSS的代码"
<body>
    <div style="background-color:red">这是DIV的区域</div>
</body>

2、内部样式表

<style>
    div {
        background-color: red;
        color: yellow;
    }
</style>

<body>
    <div style="background-color: blue;color: black;">这是div区域1</div>
    <div>这是div区域2</div>
</body>

3、导入方式

编写一个html外部的CSS文件
新建 div1.css 文件
div {
    font-family: 隶书;
    color: red;
}
html中引入 .css 文件
<style>
    @import url("div1.css");
</style>

<body>
    <div>这是div区域2</div>
</body>

4、外部样式表

引入 <link> 标签属性
新建 div1.css 文件
div {
    font-family: 隶书;
    color: red;
}

<head>
    <title>标题</title>
    <link rel="stylesheet" type="text/css" href="div1.css">
</head>

css选择器

1、标签名选择器

div{
    css代码;
}
例:span{}、table{}
标签名相同，格式相同

2、类选择器

.he{
    css代码;
}
例:
<head>
    <style type="text/css">
        .he{
            color:red;
        }
    </style>
</head>
<body>
    <div class="he">div的区域</div>
</body>

3、ID选择器

#as{
    css代码
}
例:
<head>
    <style>
        #as{
            color:red;
        }
    </style>
</head>
<body>
    <div id="as">div区域</div>
</body>

JavaScript 是一门程序设计语言，是基于对象和事件驱动的脚本语言
<script></script>标签可以放在页面的任意位置

标识符

由字母、数字、_、$组成，不能以数字开头，不能是关键字或保留字，大小写敏感

提示框

alert();
例1.
<script>
    var num = 4.56;

## 算数运算符 +、-、*、/、%
```html
数字字符串做-时，js解析器默认的将字符串解析成数字，然后再进行运算。
<script>
    alert("12" + 1);  //121
    alert("12" - 1);  //11
</script>
含有字母的字符串做+、-时，弹出NaN(非法的数字)
<script>
    alert("abc" - 1);	//NaN
</script>
布尔类型做+、-运算时。0或者null代表false；非0或者非null代表true，默认用1表示。即非0数字代表1。
<script>
    alert(true + 1); //2
    alert(false + 1); //1
</script>

比较运算符

==:只比较值是否相等
===:比较值和数据类型

<script>
    var x = 5;
    var y = "5"
    alert(x == y);	 //true
    alert(x === y);  //false
</script>

JavaScript 和 Java 的不同

1、JavaScript是基于对象的
   Java是面向对象的。
2、Javascript只需解析就可以执行
   Java需要先编译成字节码文件，然后再执行。
3、JavaScript是一种弱类型语言
   Java是强类型语言。

js语言的组成

1、ECMAScript标准
2、BOM (浏览器对象模型)
3、DOM (文档对象模型)

JS 为什么要和 HTML 结合到一起

HTML是用来封装数据。CSS通过标签设置样式。JS通过对象和语句 (循环、if) 来操作HTML标签。所有说，HTML、CSS、JS三种合在一起使用，就组成了现在看到的n多个效果。

盒子模型

 一个div 就相当于一个盒子

border：边框
padding：内边距
margin：外边距
float:漂浮属性
float:right。该元素右对齐，其下面的元素填充到该元素的左边。
float:left。该元素左对齐，其下面的元素和该元素重叠。因为该元素为float（漂浮）状态。会产生覆盖的效果。

get和post的区别

1、get方式进行提交时，参数列表内容以明文形式显示到地址栏上。
   post方式进行提交时，封装到请求体中。
2、get方式提交时，安全级别比较低。
   post方式提交时，安全级别比较高。
3、get方式提交时，对参数列表的数据要求有限制。地址栏大小是一定的，默认1KB。
   post方式提交时，对大数据没有限制。支持大数据的传输。
