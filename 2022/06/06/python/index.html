输出函数
print()

输出到文件中
ip=open('E:\study\py\测试.txt','a+')

print('hello world',file=ip)
ip.close

print(a,b,c)  # , 以一个空格间隔输出 a b c

输出字符的数据类型
print(type(a))

转义字符

\ + 首字母
\n 换行
\r 回车
例
  print("hello\nworld")
  结果
  hello
  world

标识符

由字母、数字、下划线_ 组成
不能以数字开头

数据类型

整数类型 int
  二进制   0b开头
  八进制   0o
  十六进制 0x

浮点数类型 float
  有时运算结果不准确
  from decimal import Decimal
  print(Decimal('1.1')+Decimal('2.2'))

  True、False

字符串类型 str

int(x) 其他数据类型转换成int类型

运算符

算数运算符
  + - * /
  /  浮点整除法 返回浮点结果
  // 整数整除法 向下取整

赋值运算符
  a,b,c=10,20,30
  print(a,b,c)  # 10 20 30

比较运算符
  比较 value(值)用 >、<、==、!=
  比较 id(标识)用 is(相等)、is not(不相等)
  结果为 True 或 False

布尔运算符
  and(同为True才为True)
  or(一个为True)
  not(取反)
  in(x在n中存在)、not in
    s='hello'
    print('a' in s)  # False

位运算符
  & 与 同1则1
  | 或 同0则0
  << 左移 移位补0
  >> 右移

优先级
  算数运算符>位运算符>比较>布尔>赋值运算符

结构

顺序结构

选择结构

if 语句

1.单分支
2.双分支结构
  if 条件表达式:
    条件执行体1
  else:
      条件执行体2
3.多分支结构
  if 条件表达式1:
      条件执行体1
  elif 条件表达式2:
      条件执行体2
  elif 条件表达式3:
    条件执行体3
4.嵌套
5.条件表达式
  x if 判断条件 else y
  条件成立输出x,否则输出y

循环结构

while 语句

while 条件表达式:
  条件执行体(循环体)

for-in语句

for 自定义的变量 in 可迭代对象:
  循环体
in 从可迭代对象里遍历  赋值给 自定义的变量

流程控制语句

break 语句
  用于结束循环结构
continue 语句
  结束当前循环，进入下一次循环

嵌套循环

内置函数 range()

用于生成一个整数序列
1.range(stop)  创建一个[0,stop-1]的整数序列
例:a=range(5)
   print(a)    # range(0, 5)
   print(list(a))   # [0, 1, 2, 3, 4]

2.range(a,b)   创建[a,b-1]
例:a=range(1,5)
   print(list(a))  # [1, 2, 3, 4]

3.range(a,b,c)   [1,b-1]相邻元素相差 c
例:x=range(1,10,2)
   print(list(x))  # [1, 3, 5, 7, 9]

列表

创建

使用方括号
  l1 = ['HELLO','WORLD']
调用内置函数 list()
  l2 = list(['hello','world'])

从 0 开始排序
可以重复
任意数据类型共存

查询

1.获取指定元素的索引
  列表.index(元素)
  l1.index(x) # 有相同元素时，只获取相同元素的第一个元素的索引
  l1.index(x,a,b) # 在索引[a,b-1]中查找

2.获取列表中的单个元素
  列表名[索引]
  l1[a]
  逆向索引尾部从 -1 开始

3.获取列表中的多个元素(切片)
  格式：列表名[start:stop:step]
        范围 [start,stop-1]
        相差(步长) step

  l1[::-1] # 列表逆序排列

判断、遍历

1.判断是否存在
  元素 in 列表名
  元素 not in 列表名

2.遍历
  for 迭代变量 in 列表名:
    操作
  例:for a in l1:
      print(a) # 换行单个输出

增加

格式:列表名.append(元素)

append() 在列表末尾添加一个元素
extend() 末尾至少添加一个元素
  extend(列表名) # 以列表形式添加
insert() 任意位置添加一个元素
  insert(索引,元素)
切片 任意位置至少添加一个元素
  l1[1:] = l2 # 从索引1开始把列表l2添加到l1

删除

格式:列表名.remove(元素)

remove()  一次删除一个元素
          重复元素只删除第一个
          不存在时输出ValueError
pop(索引)   删除指定索引位置上的以元素
            不指定索引，删除最后一个元素
切片    一次至少删除一个元素
        new_l1 = l1[a:b] # 删除[a,b-1]内的元素
        l1[1:3] = [] # 不产生新的列表
clear()   清空列表 # []
del()   删除列表

修改

l1[索引] = 新元素
l1[a:b] = [1, 2, 3] # [a,b-1]换成新的元素

排序

l1.sort()  从小到大排序
l1.sort(reverse=True)  从大到小排序
内置函数
  new_l1 = sorted(l1)  从小到大
