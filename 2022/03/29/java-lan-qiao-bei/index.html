Eclipse

创建Java项目

[src]右击 → [New] → [Package] → [Finish]

右击新建的包 → [New] → [Class] → 输入文件名 → 勾选”public static void main(String[] args)” → [Finish] → 完成

修改类名

右击 → [Refactor] → [Rename] → 输入新名 → [Finish]

Java

package test;

//public表示公开的
//class表示定义一个类
//hello表示一个类名
public class hello {// 表示定义一个公开的类，名为hello

	// 类体
	public static void main(String[] args) {// 表示定义一个公开的静态的主方法
		// TODO Auto-generated method stub
		System.out.println("Hello World!");// 向控制台输出一段消息
		// java中所有的"字符串"都使用双引号括起来
		System.out.println("你好！");
	}

}

标识符

由“数字、字母、下划线_、美元符号$”组成。
第一个符号不能是数字
严格区分大小写
关键字不能做标识符

运算符

取余

a % b
10 % 3 = 1

三元运算符

表达式1 ? 表达式2:表达式3
表达式1必须是boolean类型的表达式，结果为 true 或 false
表达式1为 true，结果就为表达式2
表达式1为 false，结果就为表达式3

变量

定义：数据类型 变量名；
变量名：合法标识符
赋值：变量名 = 字面值
例：int a = 1;

数据类型

基本数据类型

  1. 整数型

    byte、short、int、long

表示范围

表示范围：long > int > short > byte
短整型 short：-32768 ~ 32767 即 (-2^15 ~ 2^15-1)
整型 int：-2147483648 ~ 2147483647 即 (-2^31 ~ 2^31-1)
长整型 long：-2^63 ~ 2^63-1

  1. 浮点型

    float、double

  2. 布尔型

    boolean

    布尔表达式：只有false (假) 和true (真)

  3. 字符型

    char

    非基本数据类型

    String 字符串

    遍历字符串

    String str = "xxx";
// 第一种
for (int i = 0; i < 0; i++) {
	System.out.println(str.charAt(i));
}
// 第二种 转换成数组类型
char[] c = str.toCharArray();
for (int i = 0; i < c.length; i++) {
	System.out.println(c[i]);
}

    int 类型转化成 String 类型

    int i = 5;
// 一
String str = String.valueOf(i);
// 二
String str = i + "";
// 三
String str = Integer.toString(i);

    比较字符串内容是否相等

    // a.equals(b);   a与b的内容是否相等
String a=new String("abc"); 
String b=new String("abc");
a.equals(b);//返回true

    Integer和int的区别

    int 是基本数据类型
Integer 是 int 的包装类
Integer 的默认初始值是 null
int 的默认初始值是 0

    小数点保留

    // 保留两位小数
System.out.println(String.format("c=%.2f", c));

    String x = s.next().toString();

String s = sc.next();

控制语句

选择结构

if语句

  1. if(比较表达式){
    java语句;
}
  2. if(比较表达式){
    语句;
}else{
    语句;
}
  3. if(比较表达式){
    java语句;
}else if(比较表达式){
    Java语句;
}else if(比较表达式){
    Java语句;
}
  4. if(比较表达式){
    java语句;
}else if(比较表达式){
    Java语句;
}else if(比较表达式){
    Java语句;
}else{
    java语句;
}
    所有的控制语句都可以嵌套
    一个分支执行，全部都会结束

    switch语句

    switch(表达式){
    case值1:
       语句体1;
       break;
    case值2:
       语句体2;
       break;

    default;
       语句;
}
    case后面只能是常量
    结束条件
    遇到 break
    执行到 switch 的右大括号

    循环结构

    for 循环

    for(初始化表达式;条件表达式;更新表达式){
    循环体;
}
    先执行初始化表达式,且只执行一次

    while 循环

    while(布尔表达式){
    循环体;
}
    false 循环结束

循环次数 0 ~ N 次

do … while 循环

do{
    循环体;
}while(布尔表达式);

结束

break;
跳出当前循环，如果是嵌套循环，则只能跳出当前这一层循环。
continue;
跳出本次循环，继续执行下一次循环。
return;
从当前的方法中退出，返回到调用该方法的语句处继续执行
返回值为void的方法，从某个判断中跳出，必须用 return

数组

一维数组

数据类型 [] 数组名 = new 数据类型[数组的长度]
int[] arr = new int[5];     //可以储存5个int类型的数据
[]:代表数组,几个中括号就代表几维数组
arr:合法的标识符
new:创建新的实体或对象

二维数组

数据类型 数组名[][] = new 数据类型[m][n]; //m为数组个数，n为数组中元素个数
数据类型[] 数组名[] = new 数据类型[m][n];
int[] y[];    二维数组
int[] x,y[];  x是一维数组,y是二维数组,int[]是一个整体
相当于    int[] x;
          int[] y[];

动态初始化

知道数组长度,由系统给出初始化值
数据类型[] 数组名 = new 数据类型[数组长度];

静态初始化

数据类型[] 数组名 = new 数据类型[]{元素1,元素2,元素3,...};
// 简写
数据类型[] 数组名 = {元素1,元素2,元素3,...};

数组的输入、输出
