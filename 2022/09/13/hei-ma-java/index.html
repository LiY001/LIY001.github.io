二维数组

1.静态初始化
    数据类型[][] 数组名 = new 数据类型[][] {{元素1,元素2,...},{元素1,元素2,...}};
简化
    数据类型[][] 数组名 = {{元素1,元素2,...},{元素1,元素2,...}};
2.动态初始化
    数据结构[][] 数组名 = new 数据类型[m][n];
    m表示二维数组中可以存放多少个一维数组
    n表示每个一维数组中可以存放多少个元素。
3.获取元素
    arr[i][j]
    i:二维数组中的第 i 个一维数组
    j：一维数组中的第 j 个元素。
4.遍历二维数组
    //外循环：遍历二维数组获取里面的每一个一维数组
    //内循环：遍历一维数组获取每一个元素。
    for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr[i].length; j++) {
            System.out.print(arr[i][j] + " ");
        }
        System.out.println();
    }

方法

1.方法是程序中最小的执行单元。
2.重复的代码、具有独立功能的代码可以使用方法。
3.可以提高代码的复用性、可以提高代码的可维护性。
4.方法必须先定义后调用。
5.方法名：驼峰命名法。

简单的方法的定义和调用

1.定义：
    public static void 方法名() {
        方法体(打包起来的代码);
    }
2.调用：
    方法名();
3.例：
public class test01 {
    public static void main(String[] args) {
        //调用方法
        playGame();
        playGame();
    }

    //定义一个方法
    public static void playGame() {
        System.out.println("打游戏");
        System.out.println("失败");
        System.out.println("关机");
    }
}

带参数的方法的定义和调用

1.定义
    public static void 方法名(参数1,参数2,参数3,...) {
        ...
    }
2.调用
    方法名(参数1,参数2,参数3,...);
3.方法调用时，参数的数量与类型必须与方法定义中小括号里的变量一一对应。(形参与实参必须一一对应)
4.形参：形式参数，指方法定义中的参数。
5.实参：实际参数，指方法调用中的参数。
6.例：
public class test03 {
    public static void main(String[] args) {
        //带参数的方法的定义与调用
        getSum(10, 30);
    }

    public static void getSum(int num1, int num2) {
        int result = num1 + num2;
        System.out.println(result);
    }
}

带返回值的方法的定义和调用

1.定义
    public static 返回值类型 方法名(参数) {
        方法体;
        return 返回值;
    }
2.调用
直接调用：
    方法名(实参);
赋值调用：
    数据类型 变量名 = 方法名(实参);
输出调用：
    System.out.println(方法名(实参));
3.例：
public class test05 {
    public static void main(String[] args) {
        //直接调用
        getSum(10, 20, 30);
        //赋值调用
        int sum = getSum(10, 20, 30);
        System.out.println(sum);
        //输出调用
        System.out.println(getSum(10, 20, 30));
    }

    public static int getSum(int num1, int num2, int num3) {
        int result = num1 + num2 + num3;
        return result;//返回给方法的调用处
    }
}

public class test11 {
    public static void main(String[] args) {
        //传递基本数据类型时，传递的是真实的数据，形参的改变，不影响实际参数的值。
        int number = 100;
        System.out.println("调用change方法前：" + number);//100
        change(number);
        System.out.println("调用change方法后：" + number);//100

        //传递引用数据类型时，传递的是地址值，形参的改变，影响实际参数的值。
        int[] arr = {10, 20, 30};
        System.out.println("调用change方法前：" + arr[1]);//20
        change(arr);
        System.out.println("调用change方法前：" + arr[1]);//200
    }

    public static void change(int number) {
        number = 200;
    }

    public static void change(int[] arr) {
        arr[1] = 200;
    }
}

private 关键字

是一个权限修饰符
可以修饰成员(成员变量和成员方法)
被 private 修饰的成员只能在本类中才能访问
针对 private 修饰的成员变量，如果需要被其他类使用，需要提供相应的操作
提供“setXxx(参数)”方法，用于给成员变量赋值，方法用 public 修饰
提供“getXxx()”方法，用于获取成员变量的值，方法用 public 修饰。
public class GirlFriend {
    //属性
    private String name;
    private int age;
    private String gander;

    //针对于每一个私有化的成员变量，都要提供get和set方法
    //set方法：给成员变量赋值
    //get方法：对外提供成员变量的值

    //作用：给成员变量name进行赋值的
    public void setName(String n) {
        name = n;
    }

    //作用：对外提供name属性的
    public String getName() {
        return name;
    }

    //age
    //setAge：给成员变量age进行赋值
    //getAge：对外提供成员变量age的值
    public void setAge(int a) {
        if (a >= 18 && a <= 50) {
            age = a;
        } else {
            System.out.println("非法数据");
        }
    }

    public int getAge() {
        return age;
    }

    //gender
    public void setGander(String g) {
        gander = g;
    }

    public String getGander() {
        return gander;
    }


    //行为
    public void sleep() {
        System.out.println("睡觉");
    }

    public void eat() {
        System.out.println("吃饭");
    }
}

public class GirlFriendTest {
    public static void main(String[] args) {
        //创建对象
        GirlFriend gf1 = new GirlFriend();
        //赋值
        gf1.setName("神里凌华");
        gf1.setAge(18);
        gf1.setGander("女");

        System.out.println(gf1.getName());
        System.out.println(gf1.getAge());
        System.out.println(gf1.getGander());

        gf1.eat();

}

构造方法

1.构造方法也叫作构造器、构造函数。
2.作用：在创建对象的时候给成员变量进行初始化。
3.格式
    修饰符 类名(参数) {
        方法体;
    }
4.特点
    方法名与类名相同，大小写也要一致
    没有返回值类型，连 void 都没有
    没有具体的返回值(不能由 return 带回结果数据)
5.执行时机
    创建对象的时候由虚拟机调用，不能手动调用构造方法
    每创建一次对象，就会调用一次构造方法
6.注
构造方法的定义
    如果没有定义构造方法，系统将给出一个默认的无参数构造方法
    如果定义了构造方法，系统将不再提供默认的构造方法。
构造方法的重载
    带参构造方法和无参数构造方法，两者方法名相同，但是参数不同，这叫做构造方法的重载。
推荐的使用方式
    无论是否使用，都手动书写无参数构造方法，和带全部参数的构造方法。
无参数构造方法：初始化对象时，成员变量的数据均采用默认值。
有参数构造方法：初始化对象时，同时可以为对象进行赋值。
例：
public class Student {
    private String name;
    private int age;

    //如果我们自己没有写任何的构造方法
    //那么虚拟机给我们加一个空参构造方法
    public Student() {
        System.out.println("看看我执行了吗？");
    }

    //
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}

public class StudentTest {
    public static void main(String[] args) {
        //创建对象
        //调用的空参构造
        //Student s=new Student();

        //调用有参构造
        Student s = new Student("张三", 23);
        System.out.println(s.getName());
        System.out.println(s.getAge());
    }
}

字符串

String

1.java.lang.String类代表字符串，定义在java.lang包中，使用的时候不用导包。
2.Java 程序中的所有字符串文字(例："abc")都为此类的对象。
3.字符串的内容是不会发生改变的，它的对象在创建后不能被更改。
4.创建String对象

例：
public class StringDemo1 {
    public static void main(String[] args) {
        //1.使用直接赋值
        String s1 = "abc";
        System.out.println(s1); //abc

        //2.使用new的方式来获取一个字符串对象
        //空参构造：可以获取一个空白的字符串对象
        String s2 = new String();
        System.out.println("@" + s2 + "!"); //@!

        //传递一个字符串，根据传递的字符串内容再创建一个新的字符串对象
        String s3 = new String("abcd");
        System.out.println(s3);//abcd

        //传递一个字符串数组，根据字符串数组的内容再创建一个新的字符串对象
        //应用：可以用来修改字符串内容
        char[] chs = {'a', 'b', 'b'};
        String s4 = new String(chs);
        System.out.println(s4);//abb

        //传递一个字节数组，根据字节数组的内容再创建一个新的字符串对象
        //应用场景：在网络当中传输的数据其实都是字节信息，一般要把字节信息转换成字符串。
        byte[] bytes = {97, 98, 99};
        String s5 = new String(bytes);
        System.out.println(s5);//abc
    }
}

StringBuilder

1.StringBuilder 可以看成是一个容器，创建之后里面的内容是可变的。
2.作用：提高字符串的操作效率。
3.构造方法：创建空白可变字符对象、根据字符串内容创建可变字符串对象
4.常用方法：添加数据并返回对象本身(任意类型数据)、反转、获取长度、StringBuilder转换成字符串
5.使用场景：字符串拼接、字符串反转
public class StringBuilderDemo3 {
    public static void main(String[] args) {
        //1.创建对象
        //空白
        StringBuilder sb = new StringBuilder();
        //根据字符串内容创建可变字符串对象
        StringBuilder sb1 = new StringBuilder("123");

        //打印
        //java在底层对其做了一下特殊处理，打印对象不是地址值而是属性值(内容)。
        System.out.println(sb); //空的
        System.out.println(sb1); //123

        //2.添加元素
        sb.append(1);
        sb.append(2.3);
        sb.append(true);
        //sb.append(1).append(2.3).append(true);
        System.out.println(sb); //12.3true

        //3.反转
        sb.reverse();//123 --- 321
        System.out.println(sb); //eurt3.21

        //4.获取长度
        int len = sb.length();
        System.out.println(len); //8

        //5.StringBuilder转换成字符串
        String str = sb.toString();
        System.out.println(str); //eurt3.21
    }
}

StringJoiner

1.StringJoiner 可以看成是一个容器，创建之后里面的内容是可变的。
2.作用：提高字符串的操作效率，而且代码编写特别简洁。
3.JDK8 出现的。
4.构造方法
    StringJoiner("间隔符号");
    StringJoiner sj = new StringJoiner("---");
    StringJoiner("间隔符号","开始符号","结束符号");
    StringJoiner sj1 = new StringJoiner("---", "[", "]");
5.成员方法
添加数据并返回对象本身
返回长度(字符个数)
返回一个字符串
6.应用场景：拼接字符串
import java.util.StringJoiner;

public class StringJoinerDemo1 {
    public static void main(String[] args) {
        //1.创建一个对象，并指定中间的间隔符号
        StringJoiner sj = new StringJoiner("---");

        //2.添加元素
        sj.add("aaa").add("bbb").add("ccc");

        //获取长度
        int len = sj.length();

        //转换成字符串
        String str = sj.toString();

        //3.打印结果
        System.out.println(sj);
        System.out.println(len);
        System.out.println(str);
    }
}
