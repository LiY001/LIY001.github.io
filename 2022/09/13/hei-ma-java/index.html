入门

CMD

1.切换磁盘
    磁盘名称+冒号
    E:--切换到E盘
2.查看当前路径下的内容
    dir
3.进入下一级目录(文件夹)
    cd 目录
4.回退到上一级目录
    cd..
5.进入多级目录
    cd 目录1\目录2\...
6.回退到磁盘根目录
    cd \
7.清屏
    cls
8.退出窗口
    exit

环境变量(系统)

安装JDK
配置JAVA_HOME 路径：E:\VS\Java\jdk
配置Path      路径：%JAVA_HOME%\bin
JRE：Java的运行环境

IDEA

快捷键

格式化代码：Ctrl + Alt + L
向下复制一下：Ctrl + D
psvm + 回车：public static void main(String[] args) { }
sout + 回车：System.out.println();
背景：Settings - Appearance & Behavior - Appearance → Background Image...
------
新建类
删除类：右击 - Delete
修改类名：右击 - Refactor - Rename - 修改 - Refactor
--------
新建模块
删除模块
修改模块
导入模块
------
关闭项目：File - Close Project
新建项目
打开项目
修改项目

控制台输出乱码

Settings → Editor → File Encodings
把 Project Encoding 设置为 GBK，另外两个设置为 UTF-8。

基础

注释

1.单行注释
    //信息
2.多行注释
    /*信息*/
3.文档注释
    /**信息*/

关键字

特定涵义的英文单词
小写
class:用于(创建/定义)一个类，后面跟随类名。

字面量

整数类型
小数类型
字符串类型
字符类型
    用单引号引起来，内容只能有一个。
    \t 制表符
布尔类型
    true、false
空类型
    null
    不能直接打印，需要("null")。

变量

1.格式    数据类型 变量名 = 数据值;
2.变量名不能重复定义。
3.变量在使用前一定要进行赋值。

数据类型

基本数据类型

整数
    byte 取值范围 -128 ~ 127
    short
    int
    long
浮点数
    float
    double
字符
    char
布尔
    boolean    true、false

引用数据类型

标识符

给类、方法、变量...起的名字
1.由数字、字母、下划线_、美元符号＄组成。
2.不能以数字开头。
3.不能是关键字。
4.区分大小写。
小驼峰命名法：方法、变量名
大驼峰命名法：类名

键盘录入

1.导包，在类定义的上面
import java.util.Scanner;
2.创建对象
Scanner sc = new Scanner(System.in);   sc是变量名
3.接受数据
int i = sc.nextInt();  i是变量名

运算符

算术运算符

+、-、*、/、%(取模、取余)
    1.求个位、十位、百位...
    int ge = x % 10;
    int shi = x / 10 % 10;
    int bai = x / 100 % 10;
ASCII码：a - 97、A - 65

自增、自减运算符

++：加1
--：减1
可以放在变量的前面或后面，单独一行时，结果一样。
参与运算时：int b = a++：先赋值后加、int b = ++a：先加后赋值

赋值运算符

=、+=、-=、*=、/=、%=

关系(比较)运算符

==、！=、>、>=、<、<=
结果是布尔类型

逻辑运算符

&：与(并且)、|：或、^：异或、！：非(取反)
短路逻辑运算符
    &&：短路与、||：短路或。 结果与&、|一样

三元运算符

格式：关系表达式？表达式1:表达式2;
结果为真，执行表达式1;...结果必须被赋值或打印。

流程控制语句

顺序结构

从上向下依次执行

分支结构

if

if(关系表达式){
    语句体;
}
----------------------
if (关系表达式){
    语句体1;
} else {
    语句体2;
}
-----------------------
if(关系表达式1){
    语句体1;
}else if(关系表达式2){
    语句体2;
}
 ...
 else{
    语句体n;
}
用于对范围的判断。

switch

switch(表达式){
    case 值1:
        语句体1;
        break;
    case 值2:
        语句体2;
        break;
    ...
    default:
        语句体n+1;
        break;
}
注：1.default 可以省略，可以写在任意位置。
    2.break 不可以省略，否则会执行全部语句。(case 穿透)。
    3.用于把有限个数据一一列举出来。

循环结构

for

for(初始化语句;条件判断语句;条件控制语句){
    循环体语句;
}

while

初始化语句;
while (条件判断语句){
    循环体语句;
    条件控制语句;
}

do…while

初始化语句;
do {
    循环体语句;
    条件控制语句;
} while (条件判断语句);
先执行，后判断。

无限循环

for(;;){
    System.out.println("吃");
}
--------
while(true){
    System.out.println("吃");
}
---------
do{
    System.out.println("吃");
}while(true);

跳转控制语句

1.结束本次循环，继续下次循环
    continue;
2.结束整个循环
    break;

数组

定义

1.
数据类型[] 数组名
2.
数据类型 数组名[]

静态初始化

数据类型[] 数组名 = new 数据类型[]{元素1,元素2,元素3,...};
简化:
数据类型[] 数组名 = {元素1,元素2,元素3,...};

动态初始化

数据类型[] 数组名 = new 数据类型[数组长度];

数组元素访问

1.获取对应元素
    数组名[索引]
2.索引(下标或角标)：从 0 开始
3.把数据存储到数组中
    数组名[索引] = 具体数据或变量;

数组的遍历

for (int i = 0; i <= arr.length - 1; i++) {
    System.out.println(arr[i]);
}
数组长度：数组名.length
快速生成数组的遍历方式
    数组名.fori

数组的地址值

在内存中的位置
System.out.println(数组名);
含义：
[I@2d98a335
[ ：表示当前是一个数组。
I ：表示数组里的元素是 int 类型的。(D：double 类型)
@ ：表示一个间隔符号。(固定格式)
2d98a335 ：数组真正的地址值。(十六进制)

类

获取一个随机数

1.导包
import java.util.Random;
2.创建对象
Random r = new Random();
3.生成随机数(整数)
int number = r.nextInt(a,b);
[a,b)
